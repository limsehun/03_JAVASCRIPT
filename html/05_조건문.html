조건문

        프로그래밍 코드가 해석되는 순서를 제어하는 제어문 중 한 종류
    작성된 조건식에 따라서 수행되는 코드를 변경하는 구문

    * 조건식 : 결과가 true 또는 false인 식
    ex) boolean 단독 작성
        비교 연산자, 논리 연산자

if(만약에...)

        
            if(조건식){
                조건식이 true인 경우 수행할 코드;}

입력 받은 수가 양수인지 아닌지 검사

if(만약에)-else(아니면)문

    
        if(조건식){
            조건식이 true인 경우 수행할 코드;
        }

        else {
            //else는 조건식을 작성하는 ()가 없다.
            조건식이 false인 경우 수행 할 코드;
        }

0~99사이의 난수를 발생시켜 홀/짝 판별하기

if - else if -else

    if : 만약에
    else if : 앞서 조건이 아니라면(false)다른 조건
    else : 앞의 조건 이외(나머지, 아니면)

    
        if(조건식){
          조건식이 true인 경우 수행;
        }
  
        else if(조건식){
          앞선 if문 조건식이 false이면서,
          현재 else if문의 조건식이 true인 경우 수행;
        }

        else if(조건식){
          //...
        }

        else{
            앞선 모든 if/else if 조건이 false인 경우 수행;
        }
